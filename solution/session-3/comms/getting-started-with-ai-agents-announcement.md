# Event Announcement: Getting Started with AI Agents

**Subject:** AI agents explained without the hype — learn how they actually work

---

Everyone's talking about AI agents, but most explanations are either too vague to be useful or assume you have a PhD. This meetup fixes that.

**Getting Started with AI Agents: From Concepts to Code** is an evening of talks that will take you from "what even is an agent?" to understanding tool use, function calling, agent loops, and human-in-the-loop patterns — all in plain language with real examples.

## What You'll Learn

- **What Are AI Agents? Demystifying the Buzzword** — Marcus Johnson (Senior Developer Advocate at CloudAI) cuts through the marketing hype and explains what agents actually are, architecturally speaking
- **Tool Use & Function Calling** — How agents interact with the real world, from API calls to database queries, with concrete code patterns
- **Designing Agents for Real Users** — Elena Vasquez (Product Manager at ToolChain AI) covers human-in-the-loop patterns so your agents work *with* people, not against them
- **Lightning Round: 3 Agent Architectures in 15 Minutes** — ReAct, plan-and-execute, and multi-agent — the fast version
