# Feature Request: Meetup Communications

## Problem
We can generate event plans and schedules, but we still manually write announcements and speaker outreach emails. This is time-consuming and inconsistent.

## Request
Add the ability for MeetupBot to:
1. **Draft event announcements** — engaging posts to attract attendees, tailored to the target audience
2. **Draft speaker outreach emails** — professional invitations referencing the speaker's specific expertise

## Acceptance Criteria
- Announcements should hook the reader in the first sentence
- Speaker outreach should reference the speaker's past work and be clear about logistics
- Both should follow the tone guidelines in CLAUDE.md
- Output should be saved to the `comms/` directory with consistent naming
