# Building Production RAG Pipelines

**You'll write more code tonight than you'll hear about it.** Four guided exercises, pair programming, and floating mentors — leave with a working RAG pipeline you built yourself.

## What's Happening

This hands-on workshop lab takes you from zero to a working RAG pipeline in 3 hours. The format is simple: short intros (5-15 min) followed by guided exercises (20-30 min), each building on the last. No hour-long lectures. No watching someone else code. You build it, you keep it.

Sarah Chen leads the exercises and provides the RAG expertise. Jordan Kim and Marcus Johnson float between tables as mentors, unblocking anyone who gets stuck. Pair programming in the final exercise means you'll push your pipeline further than you could alone.

## The Lab

1. **Document Loading & Chunking** — get your data pipeline-ready
2. **Embeddings & Vector Store** — turn text into searchable vectors
3. **Query Pipeline & Response Generation** — wire retrieval to an LLM
4. **Optimize & Experiment** — pair up and push your pipeline further

Each exercise has checkpoint code, so even if you fall behind, you can jump back in at the next stage.
