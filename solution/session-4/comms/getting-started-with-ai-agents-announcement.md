# Getting Started with AI Agents: From Chatbots to Autonomous Tools

**Your chatbot is just sitting there waiting for input. Agents go out and get things done.** Join us for a beginner-friendly evening where you'll learn what AI agents actually are, see them in action, and build one yourself.

## What You'll Get

AI agents are the next evolution beyond chatbots — they use tools, make decisions, and take actions autonomously. But getting started can feel overwhelming. This workshop strips away the complexity and gives you a clear on-ramp.

You'll hear Marcus Johnson explain the fundamentals, build your first agent in a hands-on workshop, see Jordan Kim run agents locally with Ollama, and get a crash course in agent security from David Okafor. You'll leave with working code and the confidence to build more.
