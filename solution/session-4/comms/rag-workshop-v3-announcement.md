# Building Production RAG Pipelines — Workshop Lab

**You'll write more code tonight than you'll hear about it.** Four guided exercises, pair programming, and floating mentors — you'll leave with a working RAG pipeline you built yourself.

## What's Happening

Forget death-by-slides. This workshop is structured as a lab: short 5-15 minute intros followed by 20-30 minute guided exercises. Each exercise builds on the last, taking you from document loading to a fully working retrieval pipeline.

Sarah Chen leads the exercises. Jordan Kim and Marcus Johnson float between tables as mentors, making sure nobody gets stuck. By the end, you'll have running code and the understanding to extend it.

## The Exercises

1. **Document Loading & Chunking** — get your data pipeline-ready (30 min)
2. **Embeddings & Vector Store** — turn text into searchable vectors (25 min)
3. **Query Pipeline & Response Generation** — wire up retrieval to an LLM (25 min)
4. **Optimize & Experiment** — pair programming to push your pipeline further (20 min)

## What You'll Walk Away With

- A working RAG pipeline you wrote and understand
- Checkpoint code at every stage so you can revisit any step
- Pair programming connections for continued learning
